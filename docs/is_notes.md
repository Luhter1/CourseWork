## общие

### 1. Экран аутентификации
- Вкладка "Вход":
   - Поле ввода `email`

   - Поле ввода `email`

### 2. Уведомления < `GET /api/notifications`
- Бейдж `количество непрочитанных` < `GET /api/notifications/unread-count`

- Список уведомлений:
   - Кнопка "Прочитано" > `PATCH /api/notifications/{id}/read`

## художник (Х)

### 4. Х: Профиль < `GET /api/artists/me`
- Секция "Личные данные":
   - Поле ввода `имя`
   - Поле ввода `фамилия`
   - Поле ввода `местоположение`
   - Текстовое поле `биография`
   - Кнопка "Сохранить" > `PUT /api/artists/me`
- Секция "Работы" < `GET /api/artists/me/works`:
   - Кнопка "Добавить работу" > Попап "Создание работы":
      - Поле ввода `название`
      - Текстовое поле `описание`
      - Выпадающий список `направление искусства` (painting, sculpture, performance, multimedia, digital_art, photo, other)
      - Выбор даты `дата создания`
      - Поле ввода `ссылки на внешние ресурсы` (множественное)
      - Кнопка "Сохранить" > `POST /api/artists/me/works`
      - Кнопка "Редактировать" > Попап "Редактирование работы":
         - (аналогично созданию)
         - Кнопка "Сохранить" > `PUT /api/artists/me/works/{id}`
         - Кнопка "Удалить" > `DELETE /api/artists/me/works/{id}`
   - Медиа "Работы" < `GET /api/artists/me/works/{id}/media`
      - Загрузка файлов `изображения` (JPEG, PNG, до 10 МБ)
      - Загрузка файлов `видео` (MP4)
      - Кнопка "Сохранить" > `POST /api/artists/me/works/{id}/media`
      - Кнопка "Удалить медиа" > `DELETE /api/artists/me/works/{id}/media/{mediaId}`
- Секция "Достижения" < `GET /api/artists/me/achievements`:
   - Кнопка "Добавить достижение" > Попап "Создание достижения":
      - Выпадающий список `тип` (education, exhibition, publication, award)
      - Поле ввода `название`
      - Текстовое поле `описание`
      - Поле ввода `ссылка`
      - Кнопка "Сохранить" > `POST /api/artists/me/achievements`
   - Список достижений:
      - Бейдж `тип` (education, exhibition, publication, award, auto)
      - Текст `название` + Ссылка `ссылка`
      - Кнопка "Редактировать" (только для type != auto) > `PUT /api/artists/me/achievements/{id}`
      - Кнопка "Удалить" (только для type != auto) > `DELETE /api/artists/me/achievements/{id}`

## администратор резиденции (АР)

### 13. АР: Профиль резиденции < `GET /api/residences`, `GET /api/residences/me`, `GET /api/residences/{id}`
- Поле ввода `наименование`
- Текстовое поле `описание`
- Поле ввода `местоположение`
- Текстовое поле `контактные данные`
- Опубликована ли программа (is_published)
- Кнопка "Отправить" > `POST /api/residences/me`
- Кнопка "Обновить" > `PUT /api/residences/me` (если заявка на валидацию в статусе rejected, при обновлении статус меняется на `pending`)

- Статус валидации резиденции `GET /api/residences/me/validation-status` (доступ только у владельца резиденции)
   - Бейдж `статус валидации` (pending, approved, rejected)
   - Текст `комментарий суперадмина` (если есть)

### 14. АР: Статистика < `GET /api/residences/me/stats`
- Блок "Резиденция":
   - Счётчик `просмотры`
   - График просмотров по времени
- Блок "Программы":
   - Таблица:
      - Текст `название программы`
      - Счётчик `просмотры`
      - Счётчик `заявки`
   - Выпадающий список `программа`
   - Диаграмма `распределение по направлениям` (pie) < `GET /api/programs/{id}/stats`
   - Диаграмма `распределение по географии` (bar)
- Кнопка "Экспорт CSV" > `GET /api/residences/me/stats/export?format=csv`
- Кнопка "Экспорт PDF" > `GET /api/residences/me/stats/export?format=pdf`

### 15. АР: Панель программ < `GET /api/residences/me/programs`, `GET /api/residences/me/validation-status`
- Кнопка "Создать программу" (активна при статусе approved) > Экран 16 (режим создания)
- Список программ (сортировка по ближайшему дедлайну):
   - Текст `название`
   - Текст `ближайший дедлайн`
   - Бейдж `статус публикации` (опубликована/не опубликована)
   - Счётчик `заявки`
   - Счётчик `эксперты`
   - Кнопка "Управление" > Экран 16

### 16. АР: Управление программой
- Кнопка "Опубликовать" / "Снять с публикации" > `POST /api/programs/{id}/publish` / `POST /api/programs/{id}/unpublish`
- Вкладка "Информация" < `GET /api/programs/{id}`:
   - Сайдбар "История изменений" < `GET /api/programs/{id}/history`:
      - Список версий:
         - Текст `дата версии`
         - Развёртывающийся список изменённых полей (клик подставляет значение в поле)
   - Поле ввода `название`
   - Текстовое поле `цели резиденции`
   - Текстовое поле `описание`
   - Выбор даты `дедлайн подачи заявок`
   - Выбор даты `дедлайн рассмотрения заявок`
   - Выбор даты `дедлайн направления информации`
   - Поле ввода `длительность` (дни)
   - Текстовое поле `условия участия`
   - Текстовое поле `информация о дальнейших действиях` (заблокировано после отправки художникам)
   - Текстовое поле `FAQ`
   - Блок "Квоты":
      - Числовое поле `количество мест`
      - Текстовое поле `баланс направлений`
      - Числовое поле `бюджет`
   - Кнопка "Сохранить" > `PUT /api/programs/{id}`
- Вкладка "Эксперты" < `GET /api/programs/{id}/experts`:
   - Кнопка "Добавить эксперта" > Попап "Выбор эксперта" < `GET /api/experts`:
      - Поле ввода `поиск` (имя, специализация)
      - Выпадающий список `специализация` (мультиселект)
      - Список экспертов:
         - Текст `ФИО`
         - Теги `специализация`
         - Кнопка "Назначить" > `POST /api/programs/{id}/experts`
   - Таблица назначенных экспертов:
      - Текст `ФИО`
      - Текст `специализация`
      - Текст `дата назначения`
      - Бейдж `статус` (оценивает/завершил/просрочил)
      - Прогресс `оценено X из Y`
      - Кнопка "Удалить" (активна если не начал или просрочка > 14 дней) > `DELETE /api/programs/{id}/experts/{expertId}`
      - Кнопка "Принудительно удалить" (при просрочке > 14 дней) > `DELETE /api/programs/{id}/experts/{expertId}/force`
- Вкладка "Заявки" < `GET /api/programs/{id}/applications/ranking`:
   - Текст `квоты` (места, баланс, бюджет)
   - Текст `дедлайн направления информации`
   - Таблица заявок (сортировка по среднему баллу, убывание):
      - Чекбокс выбора
      - Текст `позиция`
      - Ссылка `имя художника` > Экран 11
      - Теги `направления`
      - Текст `средний балл`
      - Текст `количество оценок`
      - Бейдж `статус`
      - Кнопка "Подробнее" > Экран 17
   - Панель отбора:
      - Счётчик `выбрано / квота мест`
      - Индикатор `соответствие балансу направлений`
      - Индикатор `соответствие бюджету`
      - Предупреждение при нарушении квот
   - Кнопка "Одобрить выбранных" > Диалог подтверждения (при нарушении квот) > `POST /api/programs/{id}/applications/approve`
   - Кнопка "Завершить отбор" > `POST /api/programs/{id}/applications/finalize-selection`
   - Кнопка "Отправить информацию о дальнейших действиях" > `POST /api/programs/{id}/send-further-actions`
